# RustVid

RustVid 是一个面向普通创作者和站点维护者的 Rust Web 视频处理后台。第一版目标是支持 2GB 级断点续传、服务端 FFmpeg 转码、MP4 默认输出、HLS/m3u8 高级输出、转码后预览、下载和手动清理历史任务。

## 当前定位

- 单用户/管理员模式，不是多租户视频托管平台。
- 默认推荐 MP4 输出，HLS/m3u8 是高级发布选项。
- 代码注释、用户可见文案和项目文档使用中文；Rust 标识符、库名和协议名保留英文。

## 运行依赖

- Rust 1.95 或更新版本。
- 系统可执行的 `ffmpeg`。
- 本地磁盘空间用于保存原视频、转码工作目录和产物。

## 默认限制

- 单文件上限：2GB。
- 总存储上限：200GB。
- 同时转码任务上限：2 个。

## 本地运行

```bash
cargo run
```

默认监听 `http://127.0.0.1:3000`，数据目录为 `data/`。可以通过 `RUSTVID_DATA_DIR` 指定数据目录。

## 验证

```bash
cargo test
```

常规测试使用 fake 转码器验证任务状态流。真实 FFmpeg 样例验证应使用短视频 fixture，避免把大型媒体文件提交到仓库。
